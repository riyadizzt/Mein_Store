# Wareneingang — Änderungen ab 20.04.2026

**Wichtig für den Tagesbetrieb:**

- **Eine gebuchte Lieferschein-Zeile kann nicht mehr bearbeitet werden.** Wenn eine Zeile falsch gebucht wurde (falsche Menge, falsche Variante, falscher Preis), muss die gesamte Lieferschein-Zeile gelöscht und neu angelegt werden. Das System verhindert Mengen-Korrekturen auf einer schon gebuchten Zeile bewusst, um Mehrfach-Buchungen auszuschließen.
- **Maximal 10.000 Stück pro Zeile.** Tippfehler wie 100.000 werden abgelehnt — bitte größere Mengen auf mehrere Zeilen aufteilen.
- **Alle Zeilen müssen fehlerfrei sein,** sonst wird der ganze Lieferschein abgelehnt. Die Fehlermeldung listet alle betroffenen Zeilen auf einmal auf — einmal korrigieren, einmal absenden.
- **Der Buchen-Button deaktiviert sich während der Verarbeitung.** Nicht doppelt klicken. Falls "Bereits gebucht" erscheint, die Seite neu laden und den Bestand prüfen.
